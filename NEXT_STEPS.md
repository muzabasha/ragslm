## 🚀 Deploy to Vercel (5 Minutes)

### Step 1: Connect to Vercel

1. Visit **https://vercel.com**
2. Click **"Sign Up"**
3. Choose **"Continue with GitHub"**
4. Authorize Vercel to access your GitHub
5. Click **"Create Team"** (or use your account)

### Step 2: Import Your Repository

1. Click **"Add New..."** → **"Project"**
2. Search for **"ragslm"**
3. Click **"Import"**
4. Vercel will detect it's a static site (no build needed)
5. Click **"Deploy"**

### Step 3: Wait for Deployment

Vercel will deploy in about 30-60 seconds. You'll see:
- ✅ Building
- ✅ Ready
- 🎉 Live URL: `ragslm-[random].vercel.app`

### Step 4: Test Your Deployment

1. Click the deployment URL
2. Test all features:
   - Click SLM, RAG, Comparison tabs
   - Play animations
   - Test controls
   - Try keyboard shortcuts
3. Test on mobile

## 📝 Optional: Custom Domain

To use a custom domain (e.g., `ragslm.edu`):

1. In Vercel, go to **Settings** → **Domains**
2. Click **"Add Domain"**
3. Enter your domain
4. Update your domain registrar's DNS settings
5. Vercel will provide exact DNS records

## 📊 Monitor Your Deployment

In Vercel Dashboard:
- **Analytics**: View page views, response times
- **Logs**: See any errors or issues
- **Deployments**: View all past deployments
- **Settings**: Configure custom domain, environment variables

## 🔄 Auto-Deploy on Updates

After deployment:
- Every time you `git push` to GitHub
- Vercel automatically redeploys
- No manual steps needed!
- Deployment takes ~30 seconds

## 🛠️ Make Updates

To update the platform:

```bash
# Make changes locally
# Edit files and test

# Commit and push to GitHub
git add .
git commit -m "Update: Add new feature"
git push origin main

# Vercel automatically deploys!
```

## 🆘 If Deployment Fails

1. Check deployment logs in Vercel dashboard
2. Look for error messages
3. Common issues:
   - Wrong branch (should be "main")
   - Repository is private (make it public)
   - File paths are incorrect

## 🎯 Advanced Deployment Options

If you prefer GitHub Pages instead:

2. Select "main" branch
3. Click "Save"
