## 🎯 What This Project Does

Interactive web application where students learn:
- **SLM**: How Sequential Learning Models process information
- **RAG**: How Retrieval-Augmented Generation combines retrieval and generation
- **Comparison**: Key differences and when to use each

**No installation needed!** Pure HTML, CSS, and JavaScript.

## 🎓 Learning Modules

### 1️⃣ Sequential Learning Model (SLM)
Shows 5 steps:
1. Input Tokenization
2. Embedding Layer
3. Attention Mechanism
4. Feed Forward
5. Output Generation

### 2️⃣ Retrieval-Augmented Generation (RAG)
Shows 6 steps:
1. Query Input
2. Retrieval
3. Ranking
4. Augmentation
5. Generation
6. Citation

### 3️⃣ Comparison
Direct side-by-side comparison of SLM vs RAG with pros/cons and detailed table.
